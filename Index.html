⚙️ Intégration API ExoSupplier

Remplace TON_API_KEY par ta clé depuis exosupplier.com/api. Ce code est à placer dans ton backend (Node.js / PHP).

// ── Backend Node.js ────────────────────────── const API_KEY = "TON_API_KEY_ICI"; // Clé depuis exosupplier.com const API_URL = "https://exosupplier.com/api/v2"; // Passer une commande async function passerCommande(serviceId, lien, quantite) { const res = await fetch(API_URL, { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ key: API_KEY, action: "add", service: serviceId, // Ex: 3124 (FB page HQ) link: lien, quantity: quantite }) }); return res.json(); // Retourne : { order: 12345 } ou { error: "..." } } // Vérifier le statut d'une commande async function statutCommande(orderId) { const res = await fetch(API_URL, { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ key: API_KEY, action: "status", order: orderId }) }); return res.json(); // Retourne : { status, remains, start_count, ... } } // Lister tous les services disponibles async function listeServices() { const res = await fetch(API_URL, { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ key: API_KEY, action: "services" }) }); return res.json(); }

FAQ

Questions fréquentes

Les services ExoSupplier utilisent des méthodes progressives et naturelles. Cependant, aucun service tiers ne peut garantir zéro risque. Évitez de commander des quantités excessives d'un coup et assurez-vous que votre compte est public.

La livraison commence dans les 30 minutes après confirmation du paiement. Le délai varie selon le service : de 5 minutes (TikTok vues) à quelques heures (abonnés haute qualité). Le délai moyen est indiqué sur chaque service.

Absolument non. Nous n'avons besoin que du lien public de votre page ou profil. Ne donnez jamais votre mot de passe à un service tiers.

Après commande, vous recevez un SMS avec les instructions de paiement (numéro Wave, Orange Money, etc.). Envoyez ensuite le reçu sur notre WhatsApp pour validation. La livraison démarre automatiquement.

Un léger drop naturel est possible. Nous offrons un refill gratuit dans les 30 jours si vos chiffres baissent significativement. Contactez le support avec votre numéro de commande.

La haute qualité = comptes plus réels, moins de drops, garantie plus longue. La qualité moyenne = livraison souvent plus rapide, prix plus bas, mais drop possible plus important. Pour une utilisation long terme, la haute qualité est recommandée.
